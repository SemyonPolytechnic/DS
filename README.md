# Учебные проекты

# Классификация
### 1. [Классификация пневмонии по рентгеновскому снимку.](https://github.com/SemyonPolytechnic/DS/blob/main/Classification%20of%20pneumonia/README.md)
Было обучено 4 разные архитектуры, после сравнения которых была выбрана архитектура googlenet и дообучена. Удалось добиться на тестовой выборке качества 0.93 по метрике F1-score(recall = 0.974, precision = 0.897)
### 2. [Классификация наличия и отсутствия медицинской маски на лице человека.](https://github.com/SemyonPolytechnic/DS/blob/main/Face%20mask%20detection/README.md)
В данном учебном проекте была обучена нейронная сеть на основе архитектуры ResNet50 и удалось добиться качества 0.93 по метрике F1-score (precision = 0.88, recall = 1.0) на тестовой выборке.
### 3. [Классификация рукописных цифр (Kaggle)](https://github.com/SemyonPolytechnic/DS/tree/main/Digit%20Recognizer%20(Kaggle))
Была реализована собственная архитектура сверточной нейронной сети в учебных целях, а затем обучена. Удалось добиться accuracy = 0.995, что позволило войти в топ 10% на Kaggle.
